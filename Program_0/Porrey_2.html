CIS 223 - JavaScript


This class is an online program taught primarily through recorded lectures and the text book, available in the bookshop or through RedShelf in Canvas.
To enroll in this class, check out the registration page in the Self Service tab of your myJJC account.
This class will start in September and go to the end of the semester in December. There will be a midterm and a final exam. There will be a quiz after every chapter and a program submission each week in the Modules section of the online website.

FAQs

What are the objectives of this course and what will you learn?

Besides the main two mentioned on the home page, you will analyze real world computational problems as Boolean systems, create code to check sytax and range values to validate user input, analyze code and check the results of computations returned by the code.




When does this class meet?

Since this course is an online program, it is at your discretion to do the class work, watch the video lectures, and respond to discussions before the due date each week. The modules are posted right before the start date each week to give ample time to finish each objective or project.





How are homework and projects graded?

Online quizzes are automatically graded and can be attempted multiple times. Discussions are graded based on criteria in the description at the top of the submission page. Projects and programs have a rubric on the module page, available through the submission portion for the week the project or program is due.





Are HTML and CSS required for this class?

It is important to understand how HTML works, as you will need to use it for this class. CSS is not necessarily needed, but does add to the appearance of pages and programs.
