CIS 223 - JavaScript

What do I expect to learn and use JavaScript for?

  • Create animations or dynamic images for web pages
  • Develop web or mobile applications
  • Create interactive websites for users or developers
  • Allow web pages to do anything from a pop up message to a whole application within a site
  • Learn basic game development or mini web scrollers or things of that nature

This class has two main objectives:

  1. Translate simple, real-world problems into programming algorithms that meet user requirements and validate input
  2. Identify, troubleshoot, and fix errors and defects in code

How will we do this?

There are many things we will learn in this class to achieve the main objectives as we go through the process of learning JavaScript. Familiarizing oneself with problem solving and learning how to utilize the tools available are a main priority of becoming proficient in JavaScript and in programming in general.




In the end...

The final goal is understanding the process of finding a way to and creating a solution to any problem that one comes upon to create pages and programs.
